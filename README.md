# Stock Option Situation Lab

Static GitHub Pages app for scanning option-chain JSON snapshots into four-leg debit structures and visualizing payoff plus time/price heatmaps.

Current dataset: SPCX Dec 15 2028 option chain manually extracted from IBKR mobile screenshots while the market was closed.

## URL

Expected project-page URL after GitHub Pages deploys:

```text
https://exisz.github.io/stock/
```

## Warning

This is a research and education tool, not a live quote or order-routing tool. Always confirm live combo bid/mid/ask in IBKR Strategy Builder before placing any order.
